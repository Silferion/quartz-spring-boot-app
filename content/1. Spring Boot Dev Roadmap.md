## 🍃 The 3-Month Software Engineering Roadmap

### Month 1: Advanced Persistence, Threading & High Availability

* **Database Read Replica**
    * Understand how read replicas are provisioned as synchronized, read-only mirrors of a primary database instance.
    * Recognize the uptime and performance benefits of offloading read-heavy query traffic away from the primary write node.
    * Establish the foundational mental model required before configuring HAProxy to route traffic across replica nodes.
* **Database Integrity & Transaction Boundaries**
    * Master transactional boundaries using `@Transactional`.
    * Deep dive into transactional propagation rules (`REQUIRED`, `REQUIRES_NEW`) to prevent data corruption.
* **Asynchronous Multi-Threading**
    * Set up non-blocking background task processing using `@Async`.
    * Configure explicitly managed thread pool executors (`ThreadPoolTaskExecutor`) to handle compute-heavy background operations safely.
* **Database Load Balancing & High Availability (HAProxy)**
    * Configure an **HAProxy** instance acting as an infrastructure traffic cop directly in front of your PostgreSQL database cluster.
    * Isolate data mutations by routing write-queries to your primary master instance on one port while utilizing round-robin strategies to distribute read-queries across multiple live **Read Replicas** on another port.
* **High-Performance Caching (Redis)**
    * Implement **Redis** caching abstraction layers directly on top of heavily queried read methods.
    * Drastically lower downstream latency and completely eliminate redundant, costly database network roundtrips.

### Month 2: Distributed Security & Event-Driven Streaming

* **Keycloak Identity Federation & Dynamic RBAC**
    * Integrate an external **Keycloak** IAM server to govern centralized user authentication and handle asymmetric cryptographic JWT issuance.
    * Develop custom Spring Security interceptor filters to parse bearer tokens, reference internal PostgreSQL dynamic lookup tables, and execute runtime **Role-Based Access Control (RBAC)** down to specific individual endpoints.
* **Decoupled Messaging Ecosystems (Apache Kafka)**
    * Transition from traditional synchronous architectures to Event-Driven Design.
    * Deploy **Apache Kafka** brokers and build resilient log producers and consumers, enabling distinct microservices to share asynchronous event packets without block-waiting on HTTP connections.

### Month 3: Cloud Infrastructure & Automated Delivery

* **Declarative CI/CD Pipelines**
    * Author automated **GitHub Actions** workflows (`.github/workflows/*.yml`) triggered on mainline code commits.
    * Orchestrate linear steps to run isolated unit tests, compile binary Java application packages, compile optimized OCI Docker images, and publish them to container storage registries.
* **Container Orchestration (Kubernetes / K8s)**
    * Graduate from standalone single Docker containers to clustered topologies using local **Minikube** environments.
    * Master operational realities of declarative K8s primitives: **Pods**, **Deployments**, and **Services**.
    * Leverage self-healing control loops to automatically spin up extra horizontal replicas under sudden load spikes and instantly swap out crashing runtime instances.
